# 10. CSS3 Inheritance & Cascading

## 10.1. 상속(Inheritance)
상속되는 프로퍼티
```
visibility
opacity
font
color
line-height
text-align
white-space
```
상속되지 않는 프로퍼티 (layout 관련)
```
width/height
margin
padding
border
box-sizing
display
background
vertical-align
text-decoration
position
top/right/bottom/left
z-index
voerflow
float

```
- 요소에 따라 상속되지 않는 것도 있음. 
ex) botton

- 명시적으로 상속 지정
```
.text-red button{
  color: inherit;
}
```
